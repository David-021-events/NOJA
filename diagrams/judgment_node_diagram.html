The Judgment Node

The unit of structure. Four functions, one accountable role.

Judgment Node
Accountable Role e.g. Head of Credit Risk
Signed Artifact §
Input data, signals, observations
1
Function

What is, or might be, the case. Does not prescribe action.

Execution
The decision is carried out — by rule, algorithm, agent, or human.
Output action taken, or input to next node

The Frame — Accountability

The fourth function isn't a step in the flow. It's the boundary of the node. Exactly one named role owns everything inside, and signs the policy that governs it.

The Flow — Prediction → Judgment → Execution

Prediction describes. Judgment decides. Execution acts. The output of one node becomes the input to the next, forming a traceable network.
